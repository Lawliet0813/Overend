# OVEREND Beta Website

這是 OVEREND 產品的 Beta 測試預告網站。

## 本地開發

```bash
# 開啟本地伺服器
python3 -m http.server 8000

# 在瀏覽器開啟
open http://localhost:8000
```

## 部署到 Vercel

使用 Vercel CLI 或 GitHub 整合自動部署。

## 檔案結構

- `index.html` - 主頁面
- `style.css` - 樣式表
- `script.js` - JavaScript 功能
- `vercel.json` - Vercel 設定

## 技術棧

- 純 HTML/CSS/JavaScript
- Liquid Glass 視覺設計
- 響應式設計
