# UI 改進完成記錄

## 版本 2.1 - 對齊修正與寫作中心優化 (2025-12-29)

### 1. 佈局與對齊修正

- **問題**：`NavigationSplitView` 造成側邊欄與主內容間有無法消除的間隙，且難以精確控制對齊。
- **解決方案**：
  - 改用 `HStack` 作為根佈局容器，精確控制 `NewSidebarView` (200px) 與主內容區域的排列。
  - 移除 macOS 預設視窗標題列 (`.windowStyle(.hiddenTitleBar)`)。
  - 在側邊欄頂部加入 "OVEREND" 品牌 Header，高度與右側 Toolbar (52px) 對齊。

### 2. 側邊欄功能增強

- **新增文獻庫**：在「文獻庫」區塊標題旁加入 `+` 按鈕。
- **互動**：點擊 `+` 彈出 Sheet，輸入名稱後建立新文獻庫。

### 3. 寫作中心 (Writer Center) 優化

- **文稿管理**：
  - `DocumentCardView` 加入刪除功能。
  - 滑鼠懸停顯示紅色垃圾桶圖示。
  - 刪除前顯示確認對話框。
- **編輯器佈局**：
  - `ProfessionalEditorView` 的 A4 畫布改為自適應寬度 (`GeometryReader`)。
  - 限制最大寬度 600px，確保在開啟引用面板時不會超出視窗或被截斷。
  - 修正背景顏色與陰影效果。

## 版本 2.0 - 全新 UI 系統 (2025-12-28)

### 1. 核心設計語言

- **主色調**：`#00D97E` (Spring Green)
- **風格**：現代化、扁平、高對比
- **字體**：系統字體，強調層級與留白

### 2. 組件重構

- **側邊欄**：移除原生風格，改用自定義 `SidebarButton` 與 `SectionHeader`。
- **文獻列表**：`ModernEntryListView` 表格化設計，支援懸停操作。
- **詳情面板**：`ModernEntryDetailView` 右側滑出式設計，整合 PDF 預覽與引用功能。

### 3. 互動優化

- **懸停效果**：按鈕與列表項目加入細微背景變化。
- **動態工具列**：根據當前視圖 (文獻庫/寫作) 自動切換標題與按鈕。
