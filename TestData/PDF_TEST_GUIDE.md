# PDF 測試檔案準備說明

由於無法自動生成真實 PDF 檔案，請手動準備以下四種類型的 PDF 進行測試：

## 📄 測試 PDF 清單

### 1. 有 DOI 的國際期刊論文

**建議來源**：
- Google Scholar 搜尋並下載任一篇英文期刊論文
- 確保 PDF 包含 DOI（通常在首頁或頁尾）

**檔名範例**：
```
international_journal_with_doi.pdf
```

**測試重點**：
- 驗證 DOI 自動提取功能
- 驗證 CrossRef API 查詢
- 檢查元資料自動填寫（作者、標題、期刊等）

---

### 2. 台灣本地期刊（無 DOI）

**建議來源**：
- 臺灣期刊論文索引系統
- 華藝線上圖書館
- 政大機構典藏

**檔名範例**：
```
台灣期刊_無DOI.pdf
```

**測試重點**：
- 驗證回退到 Apple Intelligence 分析
- 檢查中文作者、標題提取
- 確認繁體中文顯示正確

---

### 3. 掃描版 PDF（測試 OCR）

**建議來源**：
- 將任一文件列印後掃描成 PDF
- 或使用手機拍照後轉 PDF

**檔名範例**：
```
scanned_document.pdf
```

**測試重點**：
- 驗證回退到 Regex 層
- 測試最低限度資訊提取
- 確認不會崩潰

---

### 4. 中文檔名 PDF

**建議來源**：
- 任何 PDF 檔案
- 重新命名為中文檔名

**檔名範例**：
```
政大碩士論文_王小明_2024.pdf
研究報告_公共行政_測試.pdf
```

**測試重點**：
- 驗證中文檔名顯示
- 確認檔名不影響匯入
- 檢查附件路徑正確

---

## 🎯 快速準備方法

### 方法一：使用現有論文

如果你有自己的論文或報告：
1. 選擇 1-2 篇已完成的論文
2. 確保至少一篇有 DOI
3. 重新命名其中一份為中文檔名

### 方法二：從網路下載

推薦網站：
- [Google Scholar](https://scholar.google.com) - 國際期刊
- [臺灣期刊論文索引](https://www.airitilibrary.com) - 本地期刊
- [政大機構典藏](https://nccur.lib.nccu.edu.tw) - 政大論文

### 方法三：建立測試 PDF

使用 macOS 內建功能：
1. 開啟任一文件（Word, Pages, 網頁）
2. 按 `Cmd + P` 列印
3. 選擇「儲存為 PDF」
4. 依照上述類型命名
