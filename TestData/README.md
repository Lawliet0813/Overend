## 🎯 使用方式

### 1. BibTeX 測試

```bash
# 在 OVEREND 中：
檔案 → 匯入 → 選擇 BibTeX 檔案
→ 選擇 TestData/*.bib
```

**建議順序**：
1. 先測試 `standard_entries.bib`（確保基本功能正常）
2. 再測試 `chinese_entries.bib`（驗證中文支援）
3. 測試 `special_chars.bib`（驗證特殊字元）
4. 最後測試 `malformed_entries.bib`（驗證容錯）

### 2. 文稿測試

**方法 A - 複製貼上**：
1. 開啟 `test_document.md` 或 `short_test.md`
2. 複製內容
3. 在 OVEREND 寫作中心建立新文稿
4. 貼上內容

**方法 B - 匯入**（如有實作）：
```bash
檔案 → 匯入 Markdown → 選擇 .md 檔案
```

### 3. PDF 測試

準備好 PDF 後：
1. 直接拖曳 PDF 到 OVEREND 視窗
2. 或使用「匯入 PDF」功能
3. 觀察 Console 輸出，確認使用哪一層提取策略
