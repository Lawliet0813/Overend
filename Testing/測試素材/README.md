# 測試素材說明

**目的：** 提供標準化測試資料，確保測試結果一致性

---

## 目錄結構

```
測試素材/
├── PDF/
│   ├── 國際期刊/          # 有 DOI，CrossRef 可查
│   ├── 台灣期刊/          # 10.6814 開頭，需 AI 萃取
│   ├── 大檔案/            # 50MB+ 測試用
│   └── 掃描版/            # 無 metadata
├── BibTeX/
│   ├── 標準格式.bib
│   ├── 含中文.bib
│   └── 格式錯誤.bib       # 測試錯誤處理
└── 文稿範本/
    └── 半成品文稿.rtf      # 展示用
```

---

## 所需素材清單

### 一、PDF 檔案

#### 1.1 國際期刊（CrossRef 可查）
**數量：** 5 篇  
**來源：** IEEE、ACM、Springer  
**要求：** 
- 有有效 DOI
- 檔案大小 2-5MB
- 包含摘要和關鍵字

**建議文獻：**
- Machine Learning 相關
- 近 5 年發表
- 英文文獻

#### 1.2 台灣期刊
**數量：** 3 篇  
**來源：** 台灣教育期刊、管理評論、資訊管理學報  
**要求：**
- DOI 前綴 10.6814 或 10.6570
- 繁體中文標題和作者
- 機構名稱為台灣大學

**測試重點：**
- Apple Intelligence 正確萃取中文
- 作者姓名順序正確
- 全形標點處理

#### 1.3 大檔案
**數量：** 1 篇  
**檔案大小：** 50MB+  
**用途：** 壓力測試

#### 1.4 掃描版 PDF
**數量：** 1 篇  
**特徵：** 無 text layer，純圖片  
**用途：** 測試降級處理

### 二、BibTeX 檔案

#### 2.1 標準格式
```bibtex
@article{example2024,
  title={Example Article Title},
  author={Smith, John and Johnson, Mary},
  journal={Example Journal},
  volume={42},
  number={3},
  pages={123--145},
  year={2024},
  doi={10.1234/example.2024.001}
}
```

#### 2.2 含中文
```bibtex
@article{chen2024,
  title={台灣高等教育政策研究},
  author={陳大明 and 李小華},
  journal={教育研究月刊},
  volume={356},
  pages={45--67},
  year={2024},
  doi={10.6814/JEPM.202401.0045}
}
```

#### 2.3 格式錯誤（測試用）
- 缺少必填欄位
- 引號不匹配
- 特殊字元

### 三、測試文稿

#### 半成品文稿內容
```
論文標題：人工智慧在教育領域的應用

一、前言
本研究探討人工智慧技術在教育領域的應用現況...

二、文獻回顧
[已插入 2-3 個引用]

三、研究方法
[待完成]

參考文獻
[自動生成]
```

**要求：**
- 1000 字左右
- 包含 3-5 個引用
- APA 格式
- 用於展示引用插入和格式切換

## 素材準備步驟

### 步驟 1：收集 PDF
1. 從學校圖書館下載 5 篇國際期刊論文
2. 從「台灣期刊論文索引系統」下載 3 篇台灣期刊
3. 找 1 篇大型 PDF（如博碩士論文）
4. 準備 1 篇掃描版 PDF

### 步驟 2：建立 BibTeX
1. 從 Google Scholar 匯出 .bib
2. 手動建立含中文的 .bib
3. 故意製作格式錯誤的 .bib

### 步驟 3：建立測試文稿
1. 在 OVEREND 建立新文稿
2. 輸入基本內容架構
3. 插入幾個引用
4. 匯出為 RTF 備份
